JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Used to assign values, compare values, perform arithmetic operations, and more.

Variables

Containers for storing data values.

Variable Naming Convention

The rules and conventions for naming your variables.

Functions

A set of statements that performs a task or calculates a value.

Methods

Actions that can be performed on objects.

Arguments

The real values passed to (and received by) the function.

Parameters

The names listed in the function definition.

Return

Ends function execution and specifies a value to be returned to the function caller.

Changing Methods

Calling a method directly on the return value of another method.

Strings

Zero or more characters written inside quotes used for storing and manipulating text.

Booleans

Data type that can only take the values true or false.

Undefined

A variable has been declared but has not yet been assigned a value.

Walrus
NaN

Not-a-Number.

Escape

The escape() function encodes a string.

Alert

Displays an alert box with a specified message and an OK button.

Comments

Single line comments start with //. Any text between // and the end of the line will be ignored by JavaScript (will not be executed).

jQuery

A JavaScript library that allows web developers to add extra functionality to their websites.

Attributes

Provides additonal information about an HTML element.
